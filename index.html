r/region

What?

r/region is a web game where you can place 1 pixel of 16 colours every minute.


Why?

I wanted to recreate r/place (a previous April fools social experiment on Reddit) more faithfully to the original than other replicas.


How?

This project is open-source and the code can be viewed here.
